Rules:

Hover on:

  • Thumbnail: It will fade away, revealing the video and a card slides down with more information about the video

  • Video: The video will start playing

  • Watch Later: the button will shrink

  • Profile picture: a popup fades in with some of the YouTuber's channel information

Hover off:

  • Thumbnail: The thumbnail will fade back in and the card will slide back up behind the video

  • Video: The video will pause then reset

  • Watch Later: The button and icon grow back to its original size

  • Profile picture: The popup fade away


Click on:

  • Watch Later: The icon changes into the "Added to Watch Later Playlist" button at 100% opacity then an animation where the icon grows then shrinks then returns to its origianl size plays

  • Remove from Watch Later: The icon changes to the "Add to Watch Later Playlist" at 50% opacity then an animation where the icon grows then shrinks then returns to its origianl size plays
